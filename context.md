## 📋 How Claude Should Use This

When this file is provided:
1. Acknowledge the project context in one sentence — do not repeat it back verbatim
2. Calibrate tone, depth, and format to the stage and ask
3. Reference the constraints and business goal when making recommendations
4. Flag any missing context that would materially change the output before starting
5. Default to the tools listed above when suggesting next steps
